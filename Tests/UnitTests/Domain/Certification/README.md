# Tests de Certificate

Este directorio contiene pruebas unitarias exhaustivas para la entidad `Certificate`.

## Cobertura

- Creación de certificados con datos válidos y nulos
- Validación de fechas de emisión y expiración
- Verificación de expiración
- Validación de errores de negocio

## Helpers

- `CertificateTestElements.cs`: contiene datos válidos reutilizables para los tests

## Ejecución

Utiliza xUnit y sigue las mejores prácticas de pruebas de dominio.
